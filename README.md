# 🏆 SEPA 雙軌強勢股選股終端機
本專案是一個基於 **Mark Minervini** 的 SEPA (Specific Entry Point Analysis) 系統所開發的台股強勢股篩選工具。程式透過結合「長線動能指標」與「短線逆風防守指標」，並配合官方即時價格修補機制，為交易者提供一個清晰的雙軌強勢股決策版面。
---
## 🌟 核心功能
1. **馬克·米奈爾維尼趨勢模板 (Trend Template)**
   * 自動檢驗個股是否滿足經典的 **Stage 2 (第二階段上升趨勢)** 七大均線過濾條件（包含 150MA > 200MA、200MA 上升滿 22 日、股價距離 52 週新高/新低比例等）。
2. **IBD 式相對強度動能評分 (Alpha RS)**
   * 先計算個股與基準（元大台灣50, `0050.TW`）的比值序列，再對比值計算三個月、六個月、九個月與十二個月的加權動能評分，精確對齊 TradingView 指標公式。
3. **短線逆風照妖鏡 (Resilience Score)**
   * 統計大盤拉回或修正期間，個股表現「抗跌並拒絕下跌」的強韌程度，篩選出在大盤回檔時有主力暗中護盤吃貨的先導股。
4. **最新行情官方 API 自動補值 (Real-time Quote Patcher)**
   * 當 yfinance 最新的交易日收盤價因為時差或延遲出現 `NaN` 時，系統會直連**台灣證券交易所 (TWSE)** 與 **證券櫃檯買賣中心 (TPEx)** 官方 API 取得今日最新收盤數據並自動補值，確保無價差干擾。
5. **基本面輔助標籤 (🔬 顯示基本面分析標籤)**
   * **Code 33 三加速**：檢測個股連續 3 季 EPS YoY、營收 YoY、淨利率是否呈現同步遞增。
   * **月營收爆發**：檢測最新單月營收是否創 12 個月新高，或 YoY 成長率連續兩月加速。

## 🛠️ 快速安裝與啟動
### 1. 安裝依賴套件
在終端機中，切換到本專案資料夾並執行以下指令安裝所需套件：
```bash
pip install -r requirements.txt
```
### 2. 啟動 Streamlit 服務
安裝完成後，執行以下指令即可在本機啟動網頁服務：
```bash
streamlit run app.py
```
啟動後，瀏覽器會自動開啟 `http://localhost:8501`。
---
## 📝 股票池輸入格式說明
* 支援在左側控制面板直接複製貼上股票清單。
* 格式非常彈性，系統內建智慧解析器，會自動過濾非代號雜訊（例如：`2330.TW,台積電` 或直接輸入 `3037`、`6187` 等純數字）。
* 上市個股請使用 `.TW` 後綴（如 `2337.TW`），上櫃個股請使用 `.TWO` 後綴（如 `8086.TWO`）。
---
## 🔌 數據連線診斷
* 側邊欄內置 API 連線診斷器，會自動回報最新月營收（FinMind）與台灣集保所的更新時間，方便確認資料庫狀態。
