# 🏆 SEPA 雙軌強勢股選股終端機

這是一個專為台股交易者設計的**強勢股篩選工具**。它基於美國投資大師馬克·米奈爾維尼 (Mark Minervini) 的 SEPA 選股心法，幫助您在大盤下跌或修正時，快速找出「跌得最少、大盤反彈時漲得最快」的潛在飆股。

---

## 🌟 這款工具能幫您做什麼？ (核心功能)

### 1. 🔍 篩選出長線多頭的強勢股 (馬克趨勢模板)
系統會自動檢查每檔股票是否符合「第二階段上升趨勢」的 7 個過濾條件（例如：股價站穩均線之上、長線均線向上、股價接近一年新高），幫您過濾掉還在下跌或盤整的弱勢股。

### 2. 📊 評估股票與大盤對比的強弱度 (相對強度分數)
採用經典的 IBD 相對強度 (RS) 計算方式（與 TradingView 社群常用的 RS 指標評分演算法一致），將個股與大盤（0050）進行對比。分數越高，代表該股票表現越超越大盤，是真正的多頭領頭羊。

### 3. 🛡️ 找出大盤下跌時的「避風港」(短線抗跌韌性)
當大盤大跌或恐慌時，這款工具會統計哪些股票「拒絕下跌」甚至逆勢上漲。這通常是法人、主力資金在暗中撐盤護盤的訊號，也是下一波多頭來臨時最容易率先噴發的股票。

### 4. 🔬 一鍵加看基本面 (營收與季報三加速)
在左側開啟基本面標籤後，系統會幫您檢查：
* **Code 33 三加速**：連續三季的 EPS 成長率、營收成長率、利潤率是否同步越賺越多。
* **營收爆發**：最新單月營收是否創一年新高，或成長速度連續兩個月加快。

### 5. 🕒 歷史回測與基準日回溯
您可以將時間調回過去的某一天（例如 2024 年大盤大跌的某天），看看當時選出的「強勢股」在後續 20 天、40 天的真實報酬率是多少，用數據驗證選股邏輯的歷史勝率。

## 💡 貼心設計 (好用在哪裡？)

* **隨便輸入都能通 (智慧輸入解析)**：
  股票輸入框非常寬容。您可以直接輸入代號（如 `2330`）、中文名稱（如 `台積電`、`大毅`），或是混合輸入（如 `2330台積電, 聯電, 3022威強電`）。支援用空格、逗號、新行隔開，系統會自動辨識並去重。
* **當日最新股價自動修補**：
  傍晚或晚上看盤時，yfinance 數據常有時差延遲。系統會自動向證交所與櫃買中心獲取今日最精確的收盤價，確保您看到的數據是最即時的。
* **API 狀態指示燈**：
  左側會顯示目前的月營收與資料庫連線是否正常，並告訴您最新數據更新到幾月，一目了然。

## 🛠️ 快速安裝與啟動

### 1. 安裝套件
請打開終端機 (Terminal)，切換到本專案資料夾，執行：
```bash
pip install -r requirements.txt
```

### 2. 啟動選股終端機
執行以下指令，系統會自動在瀏覽器打開網頁介面：
```bash
streamlit run perform.py
```
啟動後會自動開啟 `http://localhost:8501`。

### 3. 滑動歷史批次回測 (獨立工具)
如果您想批量統計所有歷史交易日的選股勝率，可以執行獨立的回測腳本：
```bash
python batch_backtest.py
```
