# 🏆 SEPA 雙軌強勢股選股終端機 (SEPA Dual-Track Decision Terminal)

本專案是一個基於 **Mark Minervini** 的 SEPA (Specific Entry Point Analysis) 核心心法所開發的台股強勢股篩選與決策終端系統。本系統結合了「技術面長線多頭模板」、「短線大盤修正抗跌韌性評分」、「相對強度動能指標 (IBD 式 Alpha RS)」，並結合 FinMind 月營收與季報三加速 (Code 33) 指標，為交易者在市場修正期間篩選出主力吃貨、抗跌且具備基本面護城河的「逆風真龍頭」與「高 Beta 攻擊兵」。

---

## 🌟 核心特色與技術架構

### 1. 🎯 雙軌強勢選股引擎
* **Mark Minervini 趨勢模板 (Stage 2)**：
  自動檢測個股是否符合經典的第二階段上升趨勢 7 大核心準則（包含 150MA > 200MA、200MA 長線均線維持上升滿 22 日、股價高於 50MA/150MA/200MA，且距離 52 週新高 25% 以內等）。
* **IBD 式相對強度分數 (Alpha RS)**：
  精確比對個股與基準大盤（元大台灣50, `0050.TW`）的比值序列，計算 3、6、9、12 個月的加權動能評分，完全對齊 TradingView 官方指標演算法。
* **短線逆風照妖鏡 (Resilience Score)**：
  在大盤回檔或修正期間，統計個股「抗跌並拒絕下跌」的強韌程度，量化主力資金在逆風中的防守意願。

### 2. ⚡ 智慧 Token 掃描器 (Smart Ticker Scanner)
* **多分隔符號自動分割**：
  輸入清單同時支援**換行 (`\n`)**、**半角逗號 (`,`)**、**全角逗號 (`，`)**、**空格 (` `)**、**Tab 鍵**、**分號 (`;`/`；`)** 作為分隔符號。
* **多格式智能解析**：
  * **純數字** (如 `2330`, `3022`) ➡️ 自動查詢資料庫並比對補上 `.TW` 或 `.TWO` 後綴。
  * **純中文名稱** (如 `台積電`, `大毅`, `威強電`) ➡️ 自動模糊與精確匹配官方標的名稱。
  * **混合格式** (如 `2330台積電`, `2478.TW`) ➡️ 透過 Regex 自動提取股票代號。
* 系統會自動過濾非代號雜訊並進行去重，輸入極具彈性。

### 3. 🔌 聯邦多線程診斷系統 (Federated Concurrent Diagnose)
* **無單點故障設計**：
  廢除綁定單一指標股（如台積電或聯電）的診斷方式。改以 `ThreadPoolExecutor` 併發查詢 **8 家台股電子龍頭巨頭**（台積電、聯電、鴻海、聯發科、廣達、光寶科、日月光、友達）的月營收。
* **第一時間同步更新**：
  擷取 8 家巨頭的最大（最新）月份。只要市場上有任何一家龍頭率先公佈最新月營收（例如聯電），儀表板便能立刻更新為最新月份（如 `2026/6`），不再被單一公司拖累。
* **30分鐘全域快取**：
  診斷結果以 `@st.cache_data(ttl=1800)` 暫存半小時，極大化節省 FinMind API 的額度消耗。

### 4. 🔬 基本面雙指標與 UI 徽章系統
* **Code 33 三加速**：
  檢測個股連續 3 季的 EPS YoY、營收 YoY、營業利益率 (或淨利率) 是否同步呈現遞增趨勢。
* **月營收爆發**：
  檢測最新單月營收是否創 12 個月新高，或 YoY 成長率連續兩月加速。
* **手機友善細節卡片**：
  在四象限明細卡片中，針對命中基本面特徵的個股顯示精美 HTML 徽章（如 `🧪 Code 33`、`🚀 月營收爆發`）。點擊 **「🔬 點擊展開詳細軌跡」** 可折疊檢視詳細數據，取代手機版上難以操作的懸停 Tooltip。

### 5. 🛡️ 安全官方行情修補 (Real-time Quote Patcher)
* **安全 SSL 連線驗證**：
  修復了關閉安全驗證的潛在中間人攻擊 (MITM) 風險，全面啟用安全的 SSL 憑證。
* **即時收盤價修補**：
  當 yfinance 最新一天的數據因時差延遲出現 `NaN` 時，系統會透過安全的官方 OpenAPI 管道直連**證交所 (TWSE)** 與 **櫃買中心 (TPEx)** API 抓取今日最新官方收盤價與成交量進行修補，確保回測與即時監控無誤差。
* **API 避退重試**：
  在請求 FinMind API 遇到 402（額度限制）或 429（速率限制）時，會自動進行 `time.sleep(1.0)` 重試避退，提升網路請求的穩定性。

### 6. 📊 獨立量化回測工具 (`batch_backtest.py`)
* 本專案包含一個獨立的命令列回測工具，可在背景下載歷史價格，每隔 20 個交易日對市場所有標的重新進行四象限分類，並統計 20 交易日後的絕對勝率、平均報酬率與相較於 0050 的超額回報。
* **Windows 編碼防護**：
  針對 Windows 控制台 (CP950) 下執行 Python 腳本可能引發的 Emoji 編碼崩潰，內置了強制 `utf-8` 編碼防護。

## 📂 專案檔案結構

* `app.py`：主 Streamlit 應用程式。包含 UI 介面、數據同步、相對強度運算、處置股修補與選股邏輯。
* `batch_backtest.py`：獨立的歷史批次量化回測腳本。
* `stocks_list.txt`：自動生成的台股代號與簡稱對照資料庫（首次執行會自動自 TWSE/TPEx OpenAPI 抓取並更新）。
* `requirements.txt`：專案依賴套件清單。
* `README.md`：專案說明文件（本檔案）。

## 🛠️ 快速安裝與啟動步驟

### 1. 安裝依賴環境
本專案建議在 Python 3.9+ 環境下執行。請在專案根目錄下，開啟終端機執行以下指令：
```bash
pip install -r requirements.txt
```

### 2. 啟動選股終端機
安裝依賴套件後，執行以下指令在本機運行 Streamlit 伺服器：
```bash
streamlit run app.py
```
啟動成功後，瀏覽器將自動開啟 `http://localhost:8501`。

### 3. 執行批次量化回測 (選擇性)
若要評估「第一象限：逆風真龍頭」與其他象限在歷史行情中的真實勝率與回報分佈，可直接在終端機中執行：
```bash
python batch_backtest.py
```
*腳本會下載歷史資料，並以 20 交易日為週期滑動統計各象限的勝率表現，並完美相容 Windows 控制台編碼。*

## 📝 控制面板使用說明

1. **股票清單輸入框**：
   * 預設已內建精選台股觀察名單（已移除低動能標的，並加入 `2478 大毅`、`3022 威強電` 等強勢股）。
   * 您可以直接在此複製貼上自訂的觀察池。支援代號與名稱混打（如 `2330, 大毅, 聯發科 2303`），系統會自動識別。
2. **🔬 顯示基本面分析標籤**：
   * 預設保持為**關閉狀態**。此時系統僅抓取 yfinance 技術面股價，**FinMind API 額度消耗為 0**，徹底消除額度焦慮。
   * 當需要深入研究象限標的時，勾選此開關並點擊執行，系統將併發拉取季報與月營收並渲染專屬徽章。
3. **🕒 歷史回溯基準日**：
   * 預設為今天（即時模式）。
   * 若將日期調回歷史某日（如 `2024-08-05` 大盤大跌日），系統將會自動**回溯時間軸**，以當日為基準日進行選股，並統計後續 N 日的真實報酬率，供您驗證在修正日選出的強勢股後續爆發力。
